Random Node Image module provide block which show x random preview images from specified node.

Recomended but not required Lightbox2 (http://drupal.org/project/lightbox2) for full view.

Instalation:
1. Copy folder with module into /sites/all/modules/
2. Enable module in Module section
3. Clear Cache
4. Go to /admin/structure/block and press configure near "Random Node Image Block" to check settings
(or direct link /admin/structure/block/manage/rni/rni_block/configure )
5. Set block position and have fun!
